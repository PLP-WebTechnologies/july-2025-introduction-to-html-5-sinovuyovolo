Understanding Semantic Tags

Semantic HTML tags provide meaning to the content, rather than just presentation. This helps browsers, screen readers, and search engines better understand the structure of a web page.

What is a <header> element?

The <header> element is used to introduce content. It's often found at the top of a page or inside other semantic elements like <article> or <section>.

Why use a <nav> element?

The <nav> element is for major navigation blocks. Using it helps screen reader users quickly find and navigate the main links on your site.
